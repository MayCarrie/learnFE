所有的 HTML 元素本质上是小的矩形块，代表着某一小块区域。有三个影响HTML元素布局的重要属性：padding(内边距)、margin(外边距)、border(边框)。
元素的 padding 控制元素内容 content和元素边框 border 之间的距离。有时想要自定义元素，使它的每一个边具有不同的 padding。CSS 允许使用 padding-top、padding-right、padding-bottom 和 padding-left来控制元素上右下左四个方向的 padding。还可以集中起来指定它们，举例如下：padding: 10px 20px 10px 20px;这四个值以顺时针方式排列：顶部、右侧、底部、左侧，简称：上右下左。
元素的外边距 margin 控制元素边框 border 和元素实际所占空间的距离。如果你将一个元素的 margin 设置为负值，元素将会变大。有时想要自定义元素，使它的每一个边具有不同的 margin。CSS 允许使用 margin-top、margin-right、margin-bottom 和 margin-left 来控制元素上右下左四个方向的 margin。还可以集中起来指定它们，举例如下：margin: 10px 20px 10px 20px;这四个值以顺时针方式排列：顶部、右侧、底部、左侧，简称：上右下左。

Hello,颜色格式


我们可以使用 6 位十六进制数字来表示颜色，每 2 位分别表示红色 (R)、绿色 (G) 和蓝色 (B) 成分。
从这三种纯色（红、绿、蓝），我们能得到 1600 万种其它的颜色。例如，橙色是纯红，混合一些绿，没有蓝。我们也可以通过平均混合所有三种颜色得到不同灰度等级的灰色。hex code 值 #808080 为灰色。
0最小,代表颜色的完全缺失。F最大，代表最大可能的亮度。所以，#000000是黑色，#FFFFFF是白色。
Hex code 遵循 red-green-blue（红-绿-蓝），或者叫 rgb 格式。hex code 中的前两位表示颜色中红色的数量，第三四位代表绿色的数量，第五六位代表蓝色的数量。所以要得到绝对的纯红色，你只需要在第一和第二位使用 F （最大可能的数值），且在第三、第四、第五和第六位使用 0 （最小可能数值）。
缩短hex code。例如，红，hex code 是 #FF0000 ，可被缩写成 #F00。也就是说，一位表示红，一位表示绿，一位表示蓝。这会把所有可能的颜色数减少至大约 4000 种，但是浏览器会把 #FF0000 和 #F00 解释为完全相同的颜色。
rgb值。代表黑色的 RGB 值看起来是下面的样子：rgb(0, 0, 0)。代表白色的 RGB 值看起来是下面的样子：rgb(255, 255, 255)。这些数值遵循 RGB 顺序模式：第一位表示红色，第二位表示绿色，第三位表示蓝色。

数据类型

JavaScript提供七种不同的data types(数据类型)，它们是undefined（未定义）, null（空）, boolean（布尔型）, string（字符串）, symbol(符号), number（数字）, and object（对象）。
通过在变量的前面使用关键字var，告诉JavaScript 来创建或者声明一个变量,就像这样：var ourName;Variable （变量）的名字可以由数字、字母、$或者_组成，大小写敏感，但是不能包含空格或者以数字为首。字符串可以用双引号或者单引号。
当 JavaScript 中的变量被声明的时候，程序内部会给它一个初始值 undefined。当你对一个值为 undefined 的变量进行运算操作的时候，比如：var a;a=a+1;算出来的结果将会是 NaN，NaN 的意思是 "Not a Number"。当你用一个没有定义的变量来做字符串连接操作的时候，比如：var a;a=a+"String";会输出'undefined String'。
驼峰命名法:变量名的第一个单词的首写字母小写，后面的单词的第一个字母大写。

构造函数

1. 声明构造函数通常使用大写字母开头，以便把自己和其他普通函数区别开。
var Car = function() {
this.wheels = 4;
this.engines = 1;
this.seats = 1;
};
在构造函数中， this指向被此构造函数创建出来的对象 。所以，当我们在 构造函数 中写：
this.wheels = 4;
这时，它创建出来的新对象将带有 wheels 属性，并且赋值为 4。可以认为 构造函数 描述了它所创建出来的对象。 2. 调用使用构造函数时，我们通过在它前面使用 new 关键字 来对它进行调用，如下：
var myCar = new Car();
myCar 现在成为了 Car 的一个 实例（instance）。现在，当 myCar （即 Car 的一个 实例 ）创建后，他可以像普通对象一样被使用，包括创建、访问、修改它的属性等，就像我们使用其他对象一样。如下：
myCar.turboType = "twin"; 我们的 myCar 变量现在有了一个 turboType 属性了，且值为 "twin" 。 注意区别与C/C++中的对象 3. 构造函数的参数
我们之前写的 构造函数 很好，但是我们不想总是创建相同的对象，怎么办呢? 为了解决这个问题，我们要向 构造函数 中添加 参数 。像下面这样：
var Car = function(wheels, seats, engines) {
this.wheels = wheels;
this.seats = seats;
this.engines = engines;
};
现在可以在调用 构造函数 时传入一组 参数 了。
var myCar = new Car(6, 3, 1);
